新しい現実への扉を開く準備はできましたか？
VRChatは、ただのゲームではありません。それは、あなたがなりたい姿になり、行きたい場所へ行ける、もう一つの現実です。
まずは、その世界に降り立つための準備を整えましょう。

1. PCスペックの確認

快適なVR体験には、相応のPCスペックが必要です。特にグラフィックボード（GPU）は重要です。

  • OS: Windows 10/11 (64bit)
  • CPU: Intel Core i5-4590 / AMD FX 8350 以上
  • Memory: 8GB以上（16GB推奨）
  • GPU: NVIDIA GeForce GTX 970 / AMD Radeon R9 290 以上

2. アカウントの作成

VRChatを始めるには、SteamアカウントとVRChatアカウントの2つを作成することをお勧めします。

Steamアカウントだけでもプレイ可能ですが、アバターのアップロードや、信頼ランク（Trust Rank）のシステムをフルに活用するためには、VRChat公式アカウントが必須となります。

公式サイト（vrchat.com）から「Registration」に進み、アカウントを作成しましょう。

3. インストールと起動

Steamストアから「VRChat」を検索し、インストールします。基本プレイは無料です。

インストールが完了したら、VRヘッドセットを接続するか、デスクトップモード（VR機器なし）で起動してみましょう。最初のチュートリアルワールドが、あなたを待っています。
